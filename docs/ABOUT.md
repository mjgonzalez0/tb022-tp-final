# Snippets

Una plataforma para compartir snippets de código.

## ¿Qué es un snippet?

Un snippet (fragmento en español) es un pequeño trozo de código reutilizable que resuelve un problema específico o realiza una tarea concreta. Por ejemplo: una función para formatear fechas o un script para validar emails. Los snippets son útiles para guardar y compartir soluciones de código que pueden reutilizarse en diferentes proyectos.
